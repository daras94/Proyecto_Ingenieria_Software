Es necesario instalar el plug-in del framework Java Play que se encuentra en la 
carpeta Plugin Play Framework en NetBeans. Ademas de esto, al abrir cualquier proyecto Play por
primera vez, NetBeans nos pedira que le indiquemos la ruta a un activador, el cual
se puede descargar en la siguiente pagina:

	https://www.lightbend.com/activator/download

Una vez descargado, descomprimir la carpeta, e indicar a NetBeans la ruta hasta 
activator-dist-1.3.12/bin
Con esto podremos ejecutar los microservicios que se encuentran en la carpeta Microservicios Independientes.

Los microservicios se conectan con una base de datos construida en MySQL, cuyo script
se encuentra en la carpeta Base de Datos MySQL y el script con los datos de prueba a insertar en la misma carpeta

Los puertos asignados a la API rest de cada microservicio se pueden consultar en el archivo
Puertos Asignados Microservicios que se encuentra en la carpeta Microservicios Independientes
